# Suphuzi VPN — Website

Official marketing and support site. Built for GitHub Pages (free hosting).

## Live URLs (after GitHub Pages setup)
- **Marketing:** `https://YOUR_USERNAME.github.io/suphuzi-vpn/`
- **Support:** `https://YOUR_USERNAME.github.io/suphuzi-vpn/support.html`

## Setup (3 steps)
1. Create GitHub repo named `suphuzi-vpn`
2. Upload all HTML files
3. Settings → Pages → Deploy from branch: main

Site goes live in 60 seconds. Free SSL included.
